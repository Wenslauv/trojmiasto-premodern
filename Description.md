этот репозиторий будет основой для публикации через сервис pages github.io
вот референс, на который я ориентировался:
https://nicklatkovich.github.io/aw-mtg/#/recent-events


Описание сайта:

отображает три вкладки (например, меню колонками слева, или еще как)
первая вкладка - вкладка игроков
вторая вкладка - вкладка ивентов
третья вкладка - вкладка колод

1. вкладка ивентов
референс - https://nicklatkovich.github.io/aw-mtg/#/recent-events

список турниров в системе

колонки такие:
1. название ивента;
2. дата
3. число игроков
4. игрок, занявший первое место
5. цвета его колоды
6. название его колоды
   
* при клике по всей строчке, кроме имени и названия колоды - открывается страница ивента (см. п4)
* при клике по имени игрока - открывается страница игрока 
* при клике по названию колоды - открывается страница колоды


2. вкладка игроков
референс - https://nicklatkovich.github.io/aw-mtg/#/players

список игроков, участвовавших в турнирах из п.1
колонки такие:
1. предпочитаемые цвета
2. имя
3. число ивентов
4. процент match wins, то есть соотношение числа выигранных матчей к числу всех сыгранных матчей

предпочитаемые цвета - пока опустим, для начала пусть будет просто пустое поле

по клику по всей строчке открывается вкладка конкретного игрока

3. вкладка конкретного игрока
референс - https://nicklatkovich.github.io/aw-mtg/#/player/brv
но с изменениями

отдельно указываются:
1. общий винрейт (число побед / число матчей)
2. любимая колода - наиболее часто играемая колода среди тех, которой игрок играл

дальше - список

колонки такие:
1. название ивента, в котором принимал участие
2. дата
3. число очков
4. место (скажем, 2/10) в ивенте
5. цвета колоды
6. название колоды
7. статистика матчей в ивенте
8. статистика игр в ивенте

формат статистики - Х-Y-Z, где X - число побед, Y - число проигрышей, Z - число ничьих, включая ID

3. вкладка конкретного ивента
референс - https://nicklatkovich.github.io/aw-mtg/#/event/442142

в целом очень похоже
отдельно указываются:
1. дата
2. число игроков
3. место проведения

дальше идет список с колонками:
1. место
2. имя
3. число очков
4. цвета колоды
5. название колоды
6. матчи в ивенте
7. игры в ивенте
8. результаты раундов в достаточном количестве

число раундов может быть любым, стоит это продумать (скроллер? перенос на две строки?), но пока тут будут дейлики максмум на 10-15 человек, то есть 4-5-6 туров

по клику по имени игрока - открывается статистика игрока
по клику по названию колоды - открывается страница колоды

Обшее:
1. страница конкретной колоды не реализовываются, оставлены для следующей итерации
2. страница всех колод не реализовывается, оставлена для следующей итерации
3. все даты указываются в формате YYYY-MM-DD
